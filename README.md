# SkeletonDemo

半自动根据页面内容生成骨架屏

## 目录

```(filePath)
├── public
│   └── shell 存放生成骨架屏文件
├── nodeScript 启动服务端代码
├── skeletonjs 生成骨架核心代码
├── ske.js 启动生成骨架
```

启动：

1. npm run serve
2. npm run start:server

port: 8080/8082
